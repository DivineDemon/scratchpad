# r0cket Plans

An automated sales department. Does product-aware lead hunting and outreach using product documentation as context. A dashboard view for monitoring, triggering and tracking the complete process.

- [ ] We need a dashboard for immediate stats
- [ ] We need to show the list of companies
- [ ] We need to show the list of contacts from each company
- [ ] We need to allow the user to add a product for which he needs to run a campaign
  - [ ] Name
  - [ ] Description
  - [ ] Documentation (PDF, Markdown, TXT)
- [ ] We need to allow the user to select the mode of campaign:
  - [ ] Manual
  - [ ] Scheduled based on user-defined frequency
- [ ] All of the information in the panel must be filtered based on the selected product. Only one product at a time.
